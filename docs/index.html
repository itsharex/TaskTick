Features功能特色

Everything you need.
Nothing you don't.

你需要的全都有，
不需要的一个没有。

Built for developers who want scheduled tasks without the complexity of system daemons.

为不想折腾系统守护进程的开发者而生。

⏱

Menu Bar Resident

菜单栏常驻

Lives in your menu bar, always running in the background. Close the main window and TaskTick keeps working — quiet, reliable, out of the way.

常驻菜单栏，后台静默运行。关闭主窗口也不影响任务执行，安静、可靠、不打扰。

📅

Cron & Interval Scheduling

Cron 表达式 & 固定间隔

Full cron expression support with visual presets and a human-readable preview. Or set simple fixed intervals. Flexible scheduling for any use case.

完整的 Cron 表达式支持，内置可视化预设和可读预览。也可设置固定间隔。灵活调度，适配各种场景。

⌨

Script Execution

脚本执行

Write scripts inline or select local files — .sh, .py, .rb, .js and more. Choose your shell, set working directory. Full control over execution.

支持内联脚本或选择本地文件 — .sh、.py、.rb、.js 等。自由选择 Shell，设定工作目录，完全掌控执行环境。

📋

Execution Logs

执行日志

Every run is logged — stdout, stderr, exit codes, and duration. Filter by status, browse history, and debug with full context.

每次执行自动记录 — 标准输出、错误输出、退出码、耗时。支持状态筛选，一目了然。

🔔

System Notifications

系统通知

Get macOS native notifications on task success or failure. Configure per task — only hear about what matters to you.

任务成功或失败时推送 macOS 原生通知。支持按任务独立配置，只关注你在意的。

🌐

Bilingual: EN & 中文

中英双语

Full English and Simplified Chinese interface. Switch languages instantly in-app or follow your system preference.

完整的中英文界面，App 内一键切换语言，或跟随系统偏好自动适配。

🔄

Auto Updates

自动更新

Automatically checks GitHub Releases for new versions. Get notified and download updates with one click.

自动检查 GitHub Releases 上的新版本，发现更新时提醒并一键下载。

✨

macOS 26 Liquid Glass

macOS 26 液态玻璃

Supports the new liquid glass visual effects on macOS 26 with graceful fallbacks for older systems. Always feels native.

支持 macOS 26 全新液态玻璃视觉特效，旧系统优雅降级。始终保持原生质感。

How It Works使用方式

Three steps to automation

三步开启自动化

1

Create a Task

创建任务

Give it a name, write your script or pick a file, choose a shell.

起个名字，编写脚本或选择文件，选择 Shell。

2

Set a Schedule

设定调度

Pick a cron preset, write a custom expression, or set a fixed interval.

选择 Cron 预设、自定义表达式，或设定固定间隔。

3

Let It Run

自动运行

TaskTick runs tasks on schedule, logs results, and notifies you. That's it.

TaskTick 按时执行任务，记录结果，通知你。就这么简单。
